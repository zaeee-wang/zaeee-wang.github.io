# JAEWANG LEE Academic Portfolio

Research-focused cinematic landing page built with React + TypeScript + Vite + Tailwind CSS.

## Local run

```bash
npm install
npm run dev
```

## Build check

```bash
npm run lint
npm run build
```

## GitHub Pages (User Site)

Use this repo as `zaeee-wang.github.io` to publish at:

`https://zaeee-wang.github.io`

The workflow file at `.github/workflows/deploy.yml` will deploy automatically when `main` is pushed.

## Connect from profile repository

In `zaeee-wang/zaeee-wang` (profile repo), add or update `README.md` with a top CTA:

```md
# JAEWANG LEE

[ENTER PORTFOLIO](https://zaeee-wang.github.io)
```

This is the closest way to make visitors entering your GitHub profile reach the portfolio immediately.
